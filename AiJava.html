Dependency Injection (DI)

A design pattern used to implement IoC. It allows the creation of dependent objects outside of a class and provides those objects to a class in different ways. In Spring, we use @Autowired to inject dependencies.

Keywords: Inversion of Control Beans ApplicationContext
